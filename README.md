## 🚀 Quick Start (10 Minutes)

**Step 1 — Get a £4/month VPS**
Sign up at [hostinger.com](https://hostinger.com) — KVM2 plan, Ubuntu 24.04.

**Step 2 — Install OpenClaw**
```bash
docker run -it --pull=always \
  -v openclaw-data:/data \
  ghcr.io/openclaw/openclaw:latest
```

**Step 3 — Get your FREE Gemini API key**
Go to [aistudio.google.com](https://aistudio.google.com) → Get API Key → Free tier = 1,500 requests/day.

**Step 4 — Give it a Soul**
Copy a SOUL.md template from the `/souls` folder. Paste your mission. Restart. Done.

**Step 5 — Connect Telegram**
Get a bot token from @BotFather. Paste into OpenClaw settings. Your agent reports to your phone.
