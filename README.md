## What is OpenClaw?

OpenClaw is a free, self-hosted AI agent framework. Give it a brain (free Gemini Flash), give it hands (browser, email, file tools), and it works autonomously while you sleep.

## 🚀 Quick Start (10 minutes)

```bash
docker run -it --pull=always \
  -v openclaw-data:/data \
  ghcr.io/openclaw/openclaw:latest
```

Get a **FREE** Gemini API key at [aistudio.google.com](https://aistudio.google.com) — 1,500 free requests/day.
